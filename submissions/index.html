Submission Formatting

This is an open call-for-papers. We invite both full papers (max 8 pages) describing mature work and short papers (max 5-6 pages) describing work-in-progress or case studies. Only original and high-quality papers submitted using IEEE 2-column format (https://www.ieee.org/conferences/publishing/templates.html), including the bibliography and any possible appendices will be considered for reviewing.
